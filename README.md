# Website

Static GitHub Pages site for Camden Lowrance.

## Quick publish path

1. Create a repository named `camden-lowrance.github.io` under the `camden-lowrance` GitHub account.
2. Push this folder to that repository.
3. In GitHub repo settings, open `Pages` and publish from the default branch root.
4. Do not set a custom domain yet.
