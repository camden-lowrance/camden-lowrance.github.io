Current Role

Beck's Hybrids

I am a Senior Geospatial Developer at Beck's. I built the current farm data pipeline from scratch and manage it day to day.

It replaced Windows servers and proprietary ArcGIS tooling with an open-source Python system built for Kubernetes-native, cloud-native operation.

If the farm data is wrong or delayed, the rest of the product has nothing solid to build on. The pipeline feeds the farm record inside FARMserver and covers harvest, planting, application, tillage, and other machine-generated field operations.

  • farm data ingestion and ETL behavior
  • geospatial processing and storage formats
  • async task execution and operational reliability
  • modernization and cleanup of older data paths
