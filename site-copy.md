## Beck's Hybrids

Kicker: Current Role

Headline: Beck's Hybrids

Body:

I am a Senior Software Engineer at Beck's. I built the current farm data pipeline from scratch and I still maintain it day to day.

It replaced Windows servers and proprietary ArcGIS tooling with open-source Python services built for Kubernetes-native, cloud-native operation.

I maintain both synchronous web APIs and asynchronous worker systems for farm data and geospatial processing, including Flask HTTP services, Celery jobs, and the related data delivery endpoints.

If the farm data is wrong or delayed, the rest of the product has nothing solid to build on. This is core infrastructure for the farm record inside FARMserver, including harvest, planting, application, tillage, and other machine-generated field operations.

Details:

- farm data ingestion and ETL behavior
- synchronous Flask API services and geospatial data delivery
- asynchronous Celery workers and long-running processing
- storage formats, reliability, and modernization of older paths
